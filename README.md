# letsupload-XSS
A simple xss, simmlar to the anonfiles xss but its letsupload.cc XSS

To deploy this project run

```bash
git clone https://github.com/DextroSec/letsupload-XSS/tree/main

cd letsupload-XSS

npm i axios commander utf8 form-data cheerio

node index.js -h 
```
